    ezt

    Welcome to ezt!

    Usage:

    ezt [OPTIONS] COMMAND [ARGS]...

    Options:

      --help  Show this message and exit.

    ezt check

    Checks if a ezt project is initiated and working.

    Usage:

    ezt check [OPTIONS]

    Options:

      --help  Show this message and exit.

    ezt init

    Create a new project in the current folder.

    Usage:

    ezt init [OPTIONS] PROJECT_NAME

    Options:

      --help  Show this message and exit.

    ezt order

    Print out the execution order of your ezt models.

    Usage:

    ezt order [OPTIONS]

    Options:

      --help  Show this message and exit.

    ezt run

    Runs the models you have created.

    Usage:

    ezt run [OPTIONS]

    Options:

      --validate          Validate yaml files before running models.
  --model-name TEXT   Provide the name of a specific model to run.
  --model-group TEXT  Provide the name of a specific model group to run all
                      models in that group and skip the rest.
  --help              Show this message and exit.
